## Capabilities

### Line mode

With empty Text field Kantomat shows branches like tree. If any symbol selected or text then it will be used as branch.

### Color modes

All color options have random radix colors 

### Export images

Png and webp formats

## Available Scripts

##### `yarn start`
##### `yarn build`
##### `yarn test`
